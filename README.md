WebChat
======================

Introduction
------------
![alt text](https://raw.githubusercontent.com/jorisbertomeu/WebChat/master/res/WebChat.png "WenChat Screen")
Simple Web Chat using Socket.IO. Multi-user, chat between them ..

Installation
------------
```sh
npm install
```

Usage
------
```sh
node server.js
```
Server will listen on port 8888 !
